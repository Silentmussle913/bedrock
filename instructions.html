Installation Instructions

üì± Mobile (Android)

  1. Download bedrock_mobile.zip from the downloads page.
  2. Use a file manager app to extract the contents.
  3. Install the provided APK file (enable "Install from unknown sources").
  4. Open the game and connect to yourserverip.com

üíª PC (Windows)

  1. Download bedrock_pc.zip from the downloads page.
  2. Extract the ZIP file to any folder.
  3. Run the file CustomMinecraft.exe inside the extracted folder.
  4. Once launched, go to multiplayer and add the server: yourserverip.com

üéÆ Console (Xbox, PS, Switch)

Unfortunately, consoles do not support custom clients. However, you can still join our server using the official Minecraft app with the following steps:

  1. Use the app Cloudflare WARP on your mobile to set up DNS tunneling (or use BedrockConnect).
  2. On your console, go to the friends tab and join the local LAN game (broadcasted from the mobile).
  3. Alternatively, use tools like BedrockConnect DNS and set the DNS to a server that supports it (more info in our Discord).
  4. Join: yourserverip.com
